# react-mini-router

这是一个学习项目，用简短的代码实现 react-router 的主要功能，用于探索路由的原理。

https://github.com/sl1673495/blogs/issues/61

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
